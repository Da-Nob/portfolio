# Portfólio Pessoal - Daniel Noberto

Portfólio desenvolvido com HTML, CSS, JavaScript e Vite.

## Como rodar

```bash
npm install
npm run dev
```

Depois abra o endereço que aparecer no terminal, normalmente `http://localhost:5173/`.

## O que trocar antes de publicar

- Links do GitHub, LinkedIn e Instagram no `index.html`.
- E-mail em `mailto:seuemail@exemplo.com` no `index.html` e no `src/main.js`.
- Links reais dos projetos nos cards da seção `Projetos`.
- Imagens dos projetos dentro da pasta `src/imgs`, se quiser substituir os placeholders.

## Build para produção

```bash
npm run build
```
